# 文章插画分支 · 只给风格,构图交给内容 + AI

**用途**:给一篇文章 / 一段内容配插画。**不套任何模板、不定版式、不固定主体**——只把「电子梦呓」的风格 DNA 注入提示词,画面画什么、怎么构图,由**文章内容 + AI 自由发挥**。产出 = 一条可直接出图的提示词(中 + 英)。

> 和 `illustration-prompts.md` 的区别:那份有固定主体公式(可爱机械吉祥物)、用来填套版里的图位;**本份只锁风格、不锁构图**,适合文章配图 / 题图 / 段落插画,让 AI 自己根据内容想画面。

## 怎么用(三步)

1. **读内容,提炼一个画面意象**:用一句话,从文章本身长出一个场景/主体(别强加固定形象)。可以多想 2~3 个备选,让 AI 各出一张挑。
2. **套提示词**:`[风格 DNA(固定)] + 画面:{你的画面意象(自由)} + [比例 / 负向词]`。
3. **出图**:交给即梦 / MJ / SD;不满意就只换「画面意象」那一句,风格 DNA 不动。

## 风格 DNA（固定不变,照抄）

复古丝网版画(risograph)质感、粗黑圆润墨线勾边、大量半调网点与点刻、有限高饱和平涂色(钴蓝 / 镉黄 / 朱红 / 品红)+ 暖米纸底、轻微纸张颗粒、可爱友好基调、平面 2D、非 3D、非写实。

## 提示词模板

**中文 · 通用 / 即梦**
```
复古丝网版画风格的可爱插画,粗黑圆润墨线,大量半调网点,有限高饱和平涂色——钴蓝、镉黄、朱红、品红,暖米色纸底,轻微纸张颗粒,可爱友好,平面 2D 非写实;画面:{从文章提炼的意象,自由构图};{比例,如 竖 3:4 / 横 16:9}
```

**EN · Midjourney / SD**
```
cute illustration in vintage risograph screenprint style, bold rounded black ink linework, heavy halftone dots, limited flat saturated palette — cobalt blue, cadmium yellow, vermilion red, hot magenta on warm cream paper, subtle paper grain, friendly and adorable, flat 2D, non-photoreal; scene: {free composition drawn from the article}; {ratio} --no 3d, photorealistic, gradient shading
```

## 示例(文章 → 自由画面)

| 文章主题 | 一句画面意象(交给 AI) |
|---|---|
| 远程办公的孤独 | 一个小机器人独自坐在漂浮的办公桌前,窗外是放大的月亮 |
| 信息过载 | 一个人被一大堆彩色信封和符号气泡温柔淹没 |
| 习惯养成 | 一株从齿轮花盆里长出的小苗,被一只手浇水 |
| 通货膨胀 | 一个购物篮里的物品越来越大、价签气球越飘越高 |

注意:同一篇可给 2~3 句不同意象,让 AI 各出一版。

## 唯一约束(可爱契约)

形象**软萌、友好、不诡异**:不要密集多眼凝视、獠牙、内脏、骷髅。可爱里带一点机械小趣味即可(详见 `style-core.md`)。除此之外,构图、主体、视角、繁简全部交给内容和 AI。

## 小贴士

- 比例随用途:公众号题图 `16:9` 或 `2:1`;竖屏 `3:4`;方图 `1:1`。
- 跑色就补:`只用 钴蓝/镉黄/朱红/品红 + 暖米底 这几种颜色`。
- 想要更统一的系列感,可固定加一句 `画面留一点暖米底空白`。
