# 风格契约 · style-core(杂志编辑 / 瑞士国际)

所有产物共享的固定内核。改套版时**照抄这里**,别自创。这套风格的灵魂 = **严格网格 + 极致留白 + 超大无衬线标题 + 单一强调色 + 细分割线**,绝对扁平、理性、克制。

## 1 · 色板(纸白 + 墨黑 + 一个强调色,强调色可换)

锁的是**结构**:浅纸底 + 近黑文字 + **最多一个**高饱和强调色(瑞士风的标志)。其余全靠黑白与留白。

**默认预设:**
```
纸白 背景 #F3F1EB   墨黑 文字/线 #111111   强调 瑞士红 #E2231A
```
**铁规:** ① 有浅纸底 ② 有近黑文字 ③ **强调色只一个**(≥2 个高饱和色就不是瑞士风了)。次级文字用墨黑降透明度(`rgba(17,17,17,.6)`)。强调色只点睛:一个词、一条线、一个块、页码。

**可换强调色预设**(只换 `--accent`):
```
瑞士红 #E2231A   克莱因蓝 #1E36C9   国际橙 #FF5400   墨绿 #1F6B4C   纯黑(无彩) #111111
```

## 2 · 字体

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Archivo:wght@600;800;900&family=Inter:wght@400;500;700&family=Noto+Sans+SC:wght@400;500;700&900&display=swap" rel="stylesheet">
```
- 大标题/数字/英文标签 = `'Archivo'`(新怪诞体,800/900);中文标题回退到 `'Noto Sans SC'` 900。统一写 `font-family:'Archivo','Noto Sans SC',sans-serif`。
- 正文 = `'Inter','Noto Sans SC',sans-serif` 400/500,行高 1.6~1.75。
- 小标签/眉标/页码:`'Archivo'` 600,**全大写 + 字间距 `letter-spacing:.12em~.2em`**。这是瑞士风的关键细节。

## 3 · 版式系统(瑞士风的骨架)

- **网格**:用 `display:grid` 排版,12 列感;内容靠**对齐**而非装饰。允许**不对称**(大留白在一侧)。
- **大边距**:页面四周留足白(长图 56px+,幻灯片 72px+)。留白是设计的一部分,不是浪费。
- **细分割线 (hairline)**:`border-top:1.5px solid #111`(或 2px)作分隔,绝不用色块阴影分隔。
- **超大字号对比**:标题极大(可到 80~140px),正文克制(15~18px)。**强对比是张力来源**。
- **编号**:章节用 `01 / 02 / 03`(Archivo)做大号索引,瑞士杂志感。

## 4 · 四个固定锚点(每个版面都要有)

1. **眉标条 (kicker)** — 顶部一行:左=全大写小标签(栏目),右=期号/日期 meta;下压一条 hairline。
2. **大标题 + 规则线** — 超大标题,**一个词**用强调色或下加强调线;标题下一条满宽 hairline。
3. **网格正文 / 编号块** — 多列对齐的正文,或 `01/02/03` 大编号 + 小标题 + 说明。
4. **页脚 meta** — 底部 hairline + 全大写小字(刊名 / 页码 / 日期)。

## 5 · 主体图形(几何 / 字体构成,不画插画)

瑞士风不靠插画,靠**几何构成 + 字体编排**:大圆、大方块、对角线、网格点,用墨黑 + 强调色平涂(无描边渐变)。需要图位时:一块 hairline 边框的留白区,放几何构成或一张去饱和图。

## 6 · 扁平铁律(本风格硬边界)

**无渐变、无阴影、无圆角(或极小)、无 3D、无手绘抖动。** 一切是平涂色块 + 直线 + 硬边。自检会拦 `*-gradient()` / `box-shadow`(非 none)/ `text-shadow`。

## 7 · 软性美学指导(理解原因,灵活执行)

- **留白即设计**:敢留大片空。瑞士风的高级感来自呼吸与秩序,塞满就垮。
- **对齐一切**:所有元素咬合网格,边缘对齐。乱一点就失去理性感。
- **强对比**:字号大小、黑白、疏密的对比,是唯一的"戏剧性"——不靠颜色堆。
- **强调色越省越贵**:一版只点一两处强调色。处处强调 = 没有强调。
