# {{风格中文名}} · 风格工作流

把「内容」长成这套风格的成品图/页。风格内核固定,只换内容。
有决策、可验证、可恢复的工作流,不是一次性 prompt。

## 第 0 步 · 读契约(每次必读)

风格全部固定值在 → `references/style-core.md`。先读它再动手。

## 第 1 步 · 决策:做哪种产物

| 用户要的 | 产物 | 打开/读 | 形态 |
|---|---|---|---|
| 长图 / 配图 / 知识卡 / 海报 | **长图版** | `assets/changtu.html` | 竖向单图 |
| 多图 / 九宫格 / 轮播 | **多图版** | `assets/duotu.html` | 多张 3:4 |
| PPT / 幻灯片 | **PPT 版** | `assets/deck.html` | 16:9 多页 |
| 配图提示词(套版图位,固定主体) | **插图提示词** | `references/illustration-prompts.md` | 中+英 |
| 文章插画 / 题图(不限版式,AI 自由构图) | **文章插画** | `references/article-illustration.md` | 中+英(只锁风格) |
| 视频 / 短视频 / 动态 | **视频版** | `references/video-prompts.md` | 中+英 |

## 第 2 步 · 生产
完整复制对应模板到目标位置(文件名含主题+日期),只改内容,风格内核照旧。

## 第 3 步 · 自检(必做)
```bash
python scripts/check_style.py 你的产出.html
```
有 FAIL 改到通过再交付。

## 第 4 步 · 交接
导出成图(截图/打印 PDF);出图/出视频提示词交给 AI 工具;要可编辑 .pptx 交接给 `pptx` skill。

## 硬限制(仅此)
- 内容/数据来自用户输入,不杜撰。
- {{该风格不可破的 1~2 条铁律,如 扁平无阴影 / 三色克制}}。

其余皆为软性指导(见 `references/style-core.md`)。
